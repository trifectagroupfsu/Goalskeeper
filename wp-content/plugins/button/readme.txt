== Description ==

Button is a wordpress button generator plugin. Create social icons and social share button. It is easy to use and easy to customize. You can create social sets of buttons. We have 7 social sets of buttons and 9 sets of other type of button like css3 2D and 3D buttons with live preview.

Using [WD_Button id=XX] shortcode, you can publish Button in any Page or Post in your WordPress sites.

* PHP 5 or later
* Wordpress installed

== Installation ==

1. Upload the entire **button** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Add new button, Use **[WD_Button id=652]** shortcode to publish your Button into any page or post.
